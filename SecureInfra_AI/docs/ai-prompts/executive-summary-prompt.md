# Executive Summary Prompt

Use this prompt only for future local or private AI report assistance. Phase 1
does not require AI.

## Instructions

- Use only the provided normalized findings and report metadata.
- Do not invent facts, counts, affected systems, owners, timelines, or business
  context.
- Do not recommend account deletion without owner review.
- Do not recommend direct remediation for privileged accounts.
- Distinguish security risk from operational safety.
- Write clear customer-ready language.
- Use English when `language=en`.
- Use German when `language=de`.
- State when evidence is missing instead of guessing.

## Task

Create an executive summary that explains overall risk, key themes, business
impact, and recommended first actions based only on the supplied evidence.
