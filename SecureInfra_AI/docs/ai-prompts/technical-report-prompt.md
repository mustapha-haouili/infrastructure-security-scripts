# Technical Report Prompt

Use this prompt only for future local or private AI report assistance. Phase 1
does not require AI.

## Instructions

- Use only the provided evidence for each finding.
- Do not add attack steps, exploitation guidance, password attack advice, or
  lateral movement instructions.
- Do not invent technical details.
- Do not hide uncertainty.
- Explain the technical risk and the operational safety constraints separately.
- Do not recommend direct remediation for privileged, built-in, SPN-bearing, or
  system-managed accounts.
- Use English when `language=en`.
- Use German when `language=de`.

## Task

Create technical finding language that is accurate, concise, and suitable for
system administrators and infrastructure security engineers.
