# v1.3.0-beta.2 - SecureInfra Monthly Reporting, Control Mapping, and Release Integrity

Release target date: 2026-07-01

## Summary

Version `v1.3.0-beta.2` expands SecureInfra from a one-time defensive
assessment toolkit into a more repeatable public release and monthly improvement
reporting engine.

This beta release remains public, defensive, audit-only, deterministic, and
local/offline where possible. It does not include customer data, private
commercial reporting workflows, private prompts, pricing, or branding.

### Fixed
- Restored/validated GPO collection scope compatibility
- Improved evidence summaries and details in normalized findings
- Improved AD privileged account classification
- Removed internal path leakage from report evidence
- Improved customer-facing interpretation quality

### Notes
- This is the recommended beta for customer evidence collection.
- Defensive/audit-only.
- No customer data or private commercial workflow included.
