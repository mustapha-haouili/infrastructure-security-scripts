# Repository Publishing Notes

These notes are maintainer-facing guidance for the public GitHub repository page. They are not part of the runtime toolkit.

## Recommended GitHub description

Enterprise infrastructure security scripts for Windows, Linux, DevSecOps, hardening, audit, monitoring, and automation.

## Suggested repository topics

```text
infrastructure-security windows-server linux powershell bash python devsecops hardening security-audit docker kubernetes automation cybersecurity
```

## Public positioning reminder

Keep the public repository focused on the defensive engine, public documentation, fictional examples, release integrity, and deterministic local reporting.

Do not add customer data, private commercial workflows, pricing, private prompts, private templates, or real assessment evidence to the public repository.
