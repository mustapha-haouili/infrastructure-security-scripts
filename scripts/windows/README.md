# Windows Scripts

These scripts focus on Windows Server and workstation security operations.

## Scripts

- `Invoke-WindowsSecurityAudit.ps1`: Collects local security baseline information and writes JSON.
- `Set-WindowsBaselineHardening.ps1`: Runs in dry-run mode by default and applies baseline changes only with `-Apply`.
- `Export-WindowsEventSecurityReport.ps1`: Exports selected Security and System event log activity.

Run PowerShell as Administrator for complete results.
