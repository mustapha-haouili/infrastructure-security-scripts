# Windows Host Scripts

Cross-host Windows scripts live here.

Use this folder for scripts that apply to both Windows Server and Windows
workstations, such as local host audits, hardening workflows, remediation plan
generation, and event security reporting.

Root-level scripts in `scripts/windows/` are compatibility wrappers that forward
to the implementations in this folder.

Planned scripts are tracked in [../../../docs/windows-roadmap.md](../../../docs/windows-roadmap.md).
