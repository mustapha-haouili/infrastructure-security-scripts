# Windows Network Scripts

Windows network scripts live here.

Use this folder for local network exposure evidence such as adapters, IP/DNS
configuration, default routes, firewall profiles, network profile category, and
listening TCP ports.

Current network scope collection writes `windows-network-exposure.json` with
findings for disabled firewall profiles, public network profiles, and sensitive
local listeners.

Planned scripts are tracked in [../../../docs/windows-roadmap.md](../../../docs/windows-roadmap.md).
