# Windows Server Scripts

Windows Server scripts live here.

Use this folder for server-focused exposure, remote management, service risk,
scheduled task, pending reboot, certificate, RDP, WinRM, and SMB audits.

Current server scope collection includes `Get-WindowsServerSecurityInventory.ps1`
for roles/features, services, scheduled tasks, SMB shares, and broad share
access evidence. It also uses cross-host scripts from `../host/` for local
administrator inventory and RDP exposure evidence.

Planned scripts are tracked in [../../../docs/windows-roadmap.md](../../../docs/windows-roadmap.md).
